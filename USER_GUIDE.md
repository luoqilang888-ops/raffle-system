# 使用说明

这份说明写给现场工作人员，不需要懂代码。

## 如何登录后台

1. 打开后台链接：`https://你的域名/admin`
2. 输入管理员邮箱和密码。
3. 点击“登录后台”。

## 如何新增组别

1. 进入活动后台。
2. 点击左侧“组别管理”。
3. 在右上角输入组别名称。
4. 点击“新增”。
5. 如需临时不参与抽奖，取消勾选“参与抽奖”。

## 如何导入名单

1. 点击左侧“参与名单”。
2. 点击“下载模板”。
3. 按模板填写编号、姓名、手机号、组别。
4. 选择“追加导入”或“覆盖导入”。
5. 点击“Excel导入”并选择文件。
6. 系统会显示成功和失败数量。

## 如何修改奖项名称

1. 点击左侧“奖项设置”。
2. 直接修改“奖项名称”输入框。
3. 调整总次数、每次组数或奖品说明。
4. 点击“保存”。

## 如何上传紧张音乐

1. 点击左侧“音乐设置”。
2. 输入音频名称。
3. 选择“紧张音乐”或“揭晓音效”。
4. 设置音量。
5. 点击“上传”。
6. 上传后可点击“试听”确认效果。

## 如何打开大屏

1. 进入活动后台首页。
2. 点击“复制大屏链接”。
3. 在会场投影电脑浏览器中打开该链接。
4. 第一次打开时，点击“点击进入会场模式并启用音效”。

## 如何开始抽奖

1. 后台首页选择当前奖项。
2. 点击“开始抽奖”。
3. 大屏会进入摇奖动画，不会提前显示结果。

## 如何停止抽奖

1. 点击“停止抽奖”。
2. 大屏会继续摇动、减速、停顿，然后揭晓抽中组别。

## 如何撤销结果

1. 点击左侧“开奖记录”。
2. 找到需要撤销的记录。
3. 点击“撤销结果”。

## 如何导出开奖记录

1. 点击左侧“开奖记录”。
2. 点击右上角“导出Excel”。
3. 浏览器会下载开奖记录文件。

## 活动前检查清单

- 后台能正常登录。
- 组别名称和参与状态已确认。
- 奖项名称、抽取次数、每次抽取组数已确认。
- 大屏链接已在投影电脑打开。
- 大屏已点击“启用音效”。
- 紧张音乐和揭晓音效已试听。
- 浏览器已进入全屏。
- 测试一次开始和停止抽奖。
- 确认大屏不显示参与者姓名、手机号或后台按钮。
